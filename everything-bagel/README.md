# Everything Bagel

One-page Next.js website for the "Everything Bagel" lifestyle and wellness channel.

## Run locally

```bash
npm install
npm run dev
```

## Notes

- Built with Next.js, React, and Tailwind CSS.
- The featured video area is a placeholder and can be replaced later with a YouTube `iframe`.
- The email form is UI-only for now and does not submit anywhere.
